Slope Plus

Slope plus

Modded version of the game Slope.
Currently on build 1.1.2

Main Features:

Seeded Runs - These won't be exactly the same as the real game, as weighted chances are implemented differently, but for the most part will work as expected. Shown on the death screen.
Custom Games - Load custom maps from the Custom Games menu, as well as set a custom starting speed, emulating the double maps glitch. Treblock practise mod is real.
Clips and Control Grabs - GONE.
(bounces are significantly reduced, but still may occur on top of tunnels and such)

Support Slopes official Publisher: Y8.com
